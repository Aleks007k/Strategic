# Analysis Result Model

## Purpose

Define the standard output structure produced by one strategic expert analysis.

AnalysisResult is the contract between expert reasoning and strategic synthesis.

## Role

AnalysisResult represents one expert's analytical contribution.

It should contain:

- what the expert concluded;
- what factors influence the outcome;
- what could go wrong;
- what could create upside;
- what assumptions support the analysis;
- how confident the expert is.
