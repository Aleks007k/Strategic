# Strategic — Personal Strategic Intelligence System

## Purpose

Strategic is a personal AI system designed to help a person think, analyze, plan and make better decisions in complex situations.

The system combines multiple AI agents with different expert roles:
- geopolitics analyst
- economist
- technology analyst
- strategist
- risk analyst
- future forecasting analyst
- business analyst

The goal is not just to answer questions, but to analyze situations, compare scenarios, estimate probabilities and help create practical strategies.

## Main principles

- Simple first version (MVP)
- Low cost
- Modular architecture
- Mobile-first approach
- AI-assisted development
- Expandable system

## Current MVP goal

Create a working personal strategic assistant that can:

1. Receive a user's question or situation.
2. Analyze it using multiple specialized AI agents.
3. Combine different perspectives.
4. Produce:
   - analysis
   - possible scenarios
   - risks
   - recommended actions
   - strategic plan
